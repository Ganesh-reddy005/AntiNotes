# AntiNotes

> **The thinking-first companion for the future of problem-solving.**

AntiNotes is an Agentic EdTech platform designed not to teach syntax, but to train logical intuition. 

In an era where AI can instantly write, debug, and optimize code, the bottleneck is no longer syntax—it is **structured thinking**. AntiNotes shifts the focus from rote memorization and pattern-matching to core cognitive reasoning.

---

## 🎯 The Core Thesis

Most EdTech platforms optimize for **content delivery and question banks**. Students solve hundreds of LeetCode problems, memorize optimal templates, and yet fail when presented with slightly modified variations in real-world scenarios or interviews. 

**AntiNotes is different.** We optimize for **retention and confidence** by teaching you how to think.

*   **No Spoons, No Shortcuts:** Instead of giving you the answers or cheat sheets, our engine guides you interactively until the solution clicks.
*   **Cognitive Profiling:** We analyze *how* you write code, your reasoning gaps, and your approach to problem-solving to build a long-term dynamic memory of your learning journey.
*   **The AI Era Shift:** As AI agents handle the boilerplate, human capability is defined by system design, decomposition, and logical patterns. AntiNotes trains the foundational mental models that remain critical when AI writes the actual code.

---

## 🚀 Core Pillars of the Platform

### 1. The Socratic Engine
The Socratic Tutor never directly hands over a solution. Instead, it analyzes your active coding session, logic flows, and compiler feedback, asking the **next best question** to nudge you in the right direction. It builds confidence through active discovery.

### 2. Cognitive Memory Loop
Unlike standard platforms that track completion states, AntiNotes compiles a **Cognitive Profile**:
*   Identifies exact reasoning mistakes (e.g., off-by-one errors, boundary neglect, state management flaws).
*   Classifies your cognitive learning style.
*   Uses spaced-repetition models to trigger timely revision milestones tailored to your past struggles.

### 3. Progressive Foundations & Practice
Our curated curriculum paths (such as **Programming Foundations**, **Foundations Logic Warm-Ups**, and **DSA Fundamentals**) ease you into complex topics with progressive coding problems, detailed hint tracks, and interactive conceptual visualizers.

---

## 🗺️ The Horizon: Beyond DSA

While our current core focus is on mastering Data Structures and Algorithms (DSA), the roadmap for AntiNotes extends far beyond:

1.  **Logical Architecture:** Moving from coding puzzles to system design, API modeling, and architectural decoupling.
2.  **AI Collaboration Patterns:** Teaching how to prompt, verify, decompose, and orchestrate systems alongside AI coders.
3.  **Fundamental Mental Models:** Abstracting logic from language-specific syntax. In the future, you won't need to learn a programming language from scratch to build software—but you will need the core logical frameworks to structure it.

## 🛠️ Tech Stack

*   **Frontend:** Next.js (App Router), TypeScript, Tailwind CSS, Framer Motion.
*   **Backend:** FastAPI (Python), MongoDB (Beanie ODM), Piston API (live code execution).
*   **Agents:** Dynamic profiling agents, memory summarizing agents, and approach-based review agents.
